#Stock Analysis Dashboard

A dynamic Stock Market Analysis Dashboard built using SQL, Python, and Power BI to analyze stock performance, risk metrics, and market trends.

This project focuses on building an end-to-end data analytics solution for stock market data. It involves:
1. Data extraction and transformation using Python
2. Structured storage using SQL
3. Interactive visualization using Power BI

The dashboard enables users to analyze stock performance across multiple dimensions such as time, sector, and individual stocks.
